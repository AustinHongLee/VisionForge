# 05-討論（未定案區）

一切還沒做完的思考都住這裡。規則五條：

1. **命名**：`YYYYMMDD-主題.md`（例：`20260710-claim-schema-草案.md`）。
2. **狀態列**：每檔頂部必須有狀態（🟦 草稿 或 🟨 討論中）與日期。
3. **無約束力**：本區文件不得作為設計或實作依據；與憲法牴觸的想法可以在這裡自由存在，但走出這個資料夾必須先過修憲程序或 ADR。
4. **畢業**：定案後，結論按性質落地為 修憲案／ADR／規劃章節；原檔標 ⬛ 已歸檔並在頂部註明「已落地於：某處」，留底不刪。
5. **登記**：長壽的討論檔（活過一週的）應登記至 docs/README 索引表。
