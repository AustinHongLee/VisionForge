## 【目標】

把「看懂」站從 stub 資料（票-0006 的 `SAMPLE_MEDIA`）換成**真後端**：匯入真圖、列出真縮圖、點開一張、輸入要找什麼、按偵測 → 把 `/infer` 回來的 `Claim` **框畫在圖上**，附概念標籤與信賴度三色標。這就是 TTFV——**使用者匯入自己的圖、十分鐘內親眼看到被框出來**。純 renderer、經 Bridge 取 API URL、不碰安全殼。

## 【範圍】（白名單）

`ui/src/renderer/**`。新增 `ui/src/renderer/src/api/`（API client）、`components/`（BoxOverlay、DetailView 等）、改「看懂」站接線與 `App.test.tsx`／新測試。**唯讀**引用 `ui/src/shared/contracts.generated`（型別）與 `window.bridge`（票-0010）。

**不得動**：`ui/src/main/**`、`ui/src/preload/**`、`ui/src/shared/**`（bridge/ipcChannels/契約已就緒，不改）、`ui/src/renderer/index.html` 的 CSP、`app/**`、`core/**`、`providers/**`、`.github/**`、`scripts/**`。

## 【介面】（結構規格——行為照此，樣式與版面自由/依 Figma）

**API client**（`ui/src/renderer/src/api/client.ts`；base URL 經 `window.bridge.getApiBaseUrl()`）：

```ts
import type { MediaRecord, Claim } from "../../../shared/contracts.generated";

interface MediaPage { items: MediaRecord[]; total: number; limit: number; offset: number; has_more: boolean; }
interface InferResult { claims: Claim[]; provider_id: string; }

listMedia(limit?: number, offset?: number): Promise<MediaPage>;      // GET /media
importFile(file: File): Promise<{ media_hash: string; deduplicated: boolean; record: MediaRecord }>; // POST /import (multipart)
thumbnailUrl(mediaHash: string): Promise<string>;                    // base + /media/{hash}/thumbnail
infer(mediaHash: string, concepts: string[]): Promise<InferResult>; // POST /infer，concepts→[{raw_text}]
```

**畫面行為**：
1. 「看懂」站載入 → `listMedia()` → 縮圖網格（`img src` 用 `thumbnailUrl`）。空專案顯示空狀態。
2. DropZone `onDrop` → 對每個檔 `importFile` → 完成後重載網格（去重的圖不重複出現）。
3. 點一張縮圖 → DetailView：顯示該圖、一個**概念輸入框**（逗號分隔，如 `bolt, crack`）＋「偵測」鈕。
4. 按偵測 → `infer(hash, concepts)` → 在圖上疊出每個 `Claim` 的 **BBox**（正規化 0–1 座標 × 顯示尺寸），框旁標 `concept.raw_text`，並依 `confidence.reliability` 上**三色**：`high`=綠、`low`=琥珀、`none`=灰（目前未接校準快照，多為 `none`＝灰，屬預期）。
5. 只畫 `geometry.type === "bbox"` 的 Claim（fixture 只出 bbox；其他型別先略過）。

## 【工作項】

1. API client（含錯誤處理：API 未就緒/網路錯 → 明確錯誤 UI，不白畫面/不吞）。

## 【驗收】（測試清單＝完成的唯一定義；mock bridge/fetch）

1. 載入時呼叫 `listMedia`，回 N 筆 → 網格出現 N 張縮圖（`img` 數＝N）。
2. DropZone drop 一檔 → 呼叫 `importFile`（帶該檔）→ 之後重載 `listMedia`。
3. 點縮圖 → DetailView 顯示；輸入 `bolt, crack` 按偵測 → 呼叫 `infer(hash, ["bolt","crack"])`。
4. `infer` 回 2 個 bbox Claim → 畫面出現 2 個框，標籤文字＝各 `raw_text`；每框有信賴度色塊（`none`→灰 class）。
5. 座標換算：mock 一個 `BBox{x1:0.1,y1:0.2,x2:0.5,y2:0.6}` → 方框樣式 left/top/width/height 依顯示尺寸成比例（斷言比例或 style）。
6. API 未就緒（`getApiBaseUrl` reject 或 fetch 500）→ 顯示錯誤狀態，不崩、不白畫面。
7. `pnpm typecheck`、`pnpm lint`、`pnpm test`、`pnpm depcruise`、`pnpm build` 全綠；renderer 無 `import electron`。
