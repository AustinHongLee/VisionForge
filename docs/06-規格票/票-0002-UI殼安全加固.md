## 【目標】
收掉票-0001 審查發現的 5 項安全加固，不新增任何功能。

## 【範圍】（白名單）
`ui/**`。**不得動 `ui/src/shared/bridge.ts`（介面凍結）、不得動 `.github/workflows/ci.yml`（既有 ui job 已涵蓋驗證）。**

## 【工作項】
1. **openExternal 協定白名單**：新增純函式 `isSafeExternalUrl(url: string): boolean`（建議放 `ui/src/main/urlPolicy.ts`），僅允許 `http:`／`https:`；main 的 handler 拒絕其他協定（拒絕時 console.warn 並靜默返回，不 throw 到 renderer）。
2. **導航鎖**：主視窗 `webContents.setWindowOpenHandler` 一律 `deny`；`will-navigate` 阻擋任何離開本 app 內容的導航。
3. **CSP**：`index.html` 加 meta CSP：`default-src 'self'; script-src 'self'; style-src 'self'; img-src 'self' data:; connect-src 'self' ws:`（`ws:` 供 dev HMR）。
4. **getPlatform fallback 留痕**：未知平台 fallback 成 `"linux"` 時 `console.warn` 記錄原始值。
5. **depcruise 規則擴充**：新增禁止 `^src/renderer` import `^src/(main|preload)`，與 `^src/(main|preload)` import `^src/renderer`（severity: error）。

## 【驗收】
1. `isSafeExternalUrl` 的 vitest 單元測試：http/https 通過；`file:`、`javascript:`、`app:`、空字串、畸形 URL 拒絕。
2. 新 depcruise 規則就位且 `pnpm depcruise` 通過。
3. `pnpm build`／`pnpm test` 全綠（CI 既有 ui job 自動覆蓋）。
4. `pnpm dev` smoke test：視窗正常、placeholder 與版本號如舊、HMR 未被 CSP 打壞。

## 【憲法】
A3（Bridge 邊界不變）、D13（depcruise 擴充）、A11 精神（失敗為常態：拒絕而非崩潰）。

## 【禁區】
不得改 Bridge 介面簽名；不得新增依賴；不得改動票面外檔案；規格衝突停手回報。
