## 【介面】（已凍結——只實作本體，不得改簽名）

```python
# app/src/visionforge_app/importer/batch.py
from dataclasses import dataclass
from pathlib import Path
from visionforge_app.importer.pipeline import ImportOutcome, DEFAULT_THUMBNAIL_MAX_EDGE

# 封閉集合，對映 MediaFormat；副檔名比對一律小寫。
SUPPORTED_SUFFIXES: frozenset[str]   # {".jpg", ".jpeg", ".png", ".webp", ".bmp", ".tif", ".tiff"}

@dataclass(frozen=True)
class BatchImportFailure:
    path: Path
    error: str      # 例外類別名稱，如 "MediaDecodeError"
    message: str

@dataclass(frozen=True)
class BatchImportOutcome:
    imported: tuple[ImportOutcome, ...]     # 新入庫（deduplicated=False）
    duplicated: tuple[ImportOutcome, ...]   # 命中既有（deduplicated=True）
    failed: tuple[BatchImportFailure, ...]  # 逐檔失敗，不中斷整批
    @property
    def total_seen(self) -> int: ...        # ＝三桶長度和（候選檔數）

def import_directory(
    project: Project,
    directory: Path,
    *,
    recursive: bool = True,
    thumbnail_max_edge: int = DEFAULT_THUMBNAIL_MAX_EDGE,
) -> BatchImportOutcome: ...
```

**消費（不修改）既有凍結介面**：`import_media(project, data, source, *, thumbnail_max_edge)`、`MediaSource(kind, detail)`、例外基類 `MediaImportError`（見 pipeline.py／errors.py）。

## 【工作項】

1. **列舉候選檔**：`recursive=True` 用 `Path.rglob`，否則 `Path.iterdir`；只收副檔名（`.suffix.lower()`）在 `SUPPORTED_SUFFIXES` 者；**非影像副檔名直接略過，不計入任何桶**（README.txt 不算失敗）。列舉結果**依路徑字串排序**後處理（確定性）。
2. **逐檔匯入**：讀 bytes → `source = MediaSource(kind="file", detail=str(path))` → `import_media(project, data, source, thumbnail_max_edge=thumbnail_max_edge)`。
3. **分桶**：`deduplicated=False` → `imported`；`True` → `duplicated`。
4. **容錯**：捕捉 `MediaImportError` 子類與檔案讀取 `OSError` → 記 `BatchImportFailure(path, error=type(e).__name__, message=str(e))`，**繼續下一檔**。不得吞掉非預期例外（讓其他例外照常拋出）。
5. `directory` 不存在或非目錄 → 拋 `NotADirectoryError`（在任何檔案處理之前）。

## 【驗收】（測試清單＝完成的唯一定義）

1. 混合資料夾（數張有效 JPG／PNG＋1 個重複內容檔＋1 個 .txt＋1 個損壞影像）→ `imported`／`duplicated`／`failed` 分桶正確；`.txt` 不出現在任何桶；`total_seen` 正確。
2. **冪等**：同資料夾連匯兩次 → 第二次全部落 `duplicated`、`project.media.count()` 不增。
3. `recursive=True` 收子目錄影像；`recursive=False` 只收頂層。
4. **確定性**：同資料夾兩次呼叫，`imported` 的順序一致（路徑排序）。
5. 損壞位元組 → 該檔進 `failed`（`error="MediaDecodeError"`），不影響同批其他檔成功入庫。
6. 空資料夾 → 三桶皆空、不拋例外。
7. `directory` 指向不存在路徑或一個檔案 → `NotADirectoryError`。
8. `python scripts/check_forbidden_imports.py`、`ruff check app`、`pytest app/tests` 全綠；`test_importer_contract.py` 維持綠。
